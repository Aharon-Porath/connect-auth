Oh-noes this is on Github!

Unfortunately as these pages are currently static you cannot try out the many bundled out of the box strategies you get with connect-auth on this server. The great news though is that if you follow these instructions then you will be able to be up and running with this example application in mere moments on your local machine:

Step 1 - Fetch the awesome

All the connect-auth code is stored over on github so we need to clone it locally (this is *NOT* required if you just want to use the library in your applications, but to experiment with available strategies or to write your own it is recommended.) 
git clone git://github.com/ciaranj/connect-auth.git
cd connect-auth

Step 2 - Update and configure

npm install -d
cp examples/example_keys_file.js examples/keys_file.js
After doing this it will be neccessary to update the keys_file to contain your secrets if you wish to use authentication providers that require them.

Step 3 - Run it!

sudo node examples/app.js
