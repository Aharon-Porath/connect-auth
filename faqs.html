How do I install connect-auth?

The easiest way to do this is with npm. From within the root of your connect or express application, execute the following command: npm install connect-auth This is all that is required to make it available, then head on over to the docs for further detais.

What license is connect-auth release under?

Why doesn't Facebook/Twitter/Yahoo work!!!?

Check your hosts! The most commonly reported issues are due to testing with local hosts then having remote providers re-direct back to a different host. As *most* strategies require the presence of a session this voids the session that the user began authenticating with and presents a brand new authentication in a 'weird' state which then fails.
